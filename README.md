# SEO Metadata Microservice (Spring Boot)

A Spring Boot microservice that analyzes SEO metadata for any given URL.

## Tech Stack

- Spring Boot (REST API)
- Spring Security (JWT)
- Hibernate / JPA (MySQL)
- Jsoup (HTML parsing)
- (Optional later) Vaadin dashboard
